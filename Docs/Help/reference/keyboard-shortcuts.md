# Keyboard Shortcuts

## Window & Application

| Shortcut | Action |
|---|---|
| **⌘,** | Open Settings |
| **⌘W** | Close window (keeps app running, preserves sessions) |
| **⌘⇧W** | Close current tab |
| **⌘Q** | Quit (confirmation shown if Direct sessions are running) |
| **⌘M** | Minimise window |
| **⌘H** | Hide application |
| **⌘?** | Open Help |

## Terminal Management

| Shortcut | Action |
|---|---|
| **⌘N** | New terminal (with dialog) |
| **⌘T** | Quick new terminal in same column and working directory |
| **⌘⇧N** | New column |
| **⌘⌫** | Delete current terminal (moves to bin) |
| **⌘⇧⌫** | Open bin |

## Navigation

| Shortcut | Action |
|---|---|
| **⌘B** | Back to board |
| **⌘K** | Command palette |
| **⌘]** | Next pinned terminal tab |
| **⌘[** | Previous pinned terminal tab |

## View & Search

| Shortcut | Action |
|---|---|
| **⌘⌥Z** | Toggle zoom mode |
| **⌘F** | Find in terminal buffer |
| **⌘⇧S** | Export session to file |

## Actions

| Shortcut | Action |
|---|---|
| **⌘D** | Toggle favourite on current terminal |
| **⌘⇧T** | Open in Terminal.app at current working directory |
